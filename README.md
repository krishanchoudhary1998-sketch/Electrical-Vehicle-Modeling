# Electrical-Vehicle-Modeling
This repository contains a complete electric vehicle propulsion system developed in MATLAB and Simulink. The work includes modeling, simulation, analysis and a detailed project report.

**Contents**

**Final_electric_vehicle_project.slx**

Full EV propulsion model with battery pack, motor controller and vehicle dynamics.

**PMSM.slx**

Individual Permanent Magnet Synchronous Motor model used for subsystem analysis.

**Report Electric vehicle modelling.pdf**

A detailed report explaining the modeling approach, assumptions, simulations and results.

**Project Overview**

The project focuses on building and testing a full EV propulsion chain. The model captures electrical, mechanical and control-level behavior to study performance under real driving conditions.

**Key Features**

A complete EV propulsion model that includes:

DC motor (PMSM model provided separately)

Battery pack

Motor controller

Vehicle longitudinal dynamics

Real drive cycle simulations for:

Energy consumption analysis

Speed tracking performance

Regenerative braking behavior

Battery performance evaluation:

SOC variation across different loads

Drivetrain response under various driving conditions

How to Use

Open the Final_electric_vehicle_project.slx files in MATLAB Simulink.

Run the drive cycle simulations included in the main model.

Refer to the PDF report for detailed explanations and interpretation of results.

**Requirements**

MATLAB

Simulink

Simscape (recommended)
